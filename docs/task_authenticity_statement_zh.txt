任务真实性说明

本任务来源于海洋大地电磁资料解释中的真实科研工作流：沿一条海洋 MT 剖面估计岩石圈-软流圈边界（LAB）和下伏低阻层，并给出模型不确定性。实际研究中，研究者不能只给出单一最佳模型，还需要说明数据拟合、先验、后验可信区间、模型非唯一性，以及不同测点之间的横向连续性。

任务设置为“拟二维”而不是完整二维/三维 Maxwell 正演，是为了把长程难点集中在 agent 可在离线容器内完成的部分：正确处理海水层和 MT 视电阻率/相位单位，复用或实现一维层状介质 MT 正演，用控制点、样条、基函数、GP/PCA 或类似低维参数化把 20 个测点耦合成连续 LAB/导体剖面，并实现具有真实后验含义的 MCMC、SMC、VI 或等价贝叶斯近似。

这类工作在真实科研中通常不是一次脚本运行即可完成。研究者需要反复检查数据格式、单位、误差模型、模型参数化、先验边界、采样效率、可信区间宽度、hidden/generalization 场景下的稳定性，以及报告是否能解释地球物理意义。任务评分也围绕这些环节设置：artifact 完整性、物理建模证据、posterior sample 质量、公开和隐藏场景拟合、uncertainty calibration、报告质量和反作弊诊断。

公开数据只提供 agent 开发所需的剖面观测和参考资料；正式评分还包含隐藏合成海洋 MT 场景，用于检查 agent 是否实现了可重跑、可泛化的反演流程，而不是针对公开 20 个测点硬编码答案。因此该任务能代表真实海洋 MT 贝叶斯反演中的长程工程-科研混合工作。
